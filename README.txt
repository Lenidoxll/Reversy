1. Без qtCreator программу можно запустить, распаковав приложенный в эосе архив.
2. Программа управляется клавиатурой.
3. В настройках игроков для редактирования необходимо нажать enter. Аналогично для сохранения установленных настроек.
4. На поле фишки ставятся после нажатия enter (как для игрока-человека, так и для игрока-компьютера).
